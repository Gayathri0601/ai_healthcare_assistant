AI Healthcare Assistant 🏥
📌 Overview

This project is an AI-powered healthcare assistant that allows users to input symptoms and receive structured health insights such as possible conditions, severity level, and recommendations.

🚀 Features
Symptom-based analysis
Severity classification (Low / Medium / High)
AI-generated suggestions
Simple web interface using Streamlit
🧠 How it Works
User enters symptoms
System classifies severity using simple rules
AI generates structured response
Output is displayed in a web interface
🛠️ Tech Stack
Python
OpenAI API
Streamlit
Ngrok
⚙️ Setup Instructions

Clone the repo:

git clone https://github.com/your-username/ai-healthcare-assistant.git

Install dependencies:

pip install -r requirements.txt

Add your API key:

export OPENAI_API_KEY=your_api_key_here

Run the app:

streamlit run app.py
